# MINDHeard

# Description
MINDHeard est un jeu vidéo 2D qui invite les utilisateurs à explorer les dynamiques sociales équines à travers la gestion d’un troupeau. Chaque cheval possède un trait de caractère spécifique, influençant les interactions et l’impact sur les statistiques du jeu. Le joueur apprend à gérer les relations et les équilibres sociaux du groupe grâce à un système de questions et de choix interactifs.

MINDHeard propose une expérience éducative mêlant réflexion et sensibilisation à la complexité des prises de décisions.
